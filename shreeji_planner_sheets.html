Your planner data will be stored in your own Google Sheet — free, private, and synced across all devices.

  2. Delete any existing code, paste the contents of Code.gs
  3. Click Deploy → New Deployment
  4. Type: Web app · Execute as: Me · Access: Anyone
  5. Click Deploy → Authorize → Copy the Web App URL
  6. Paste it below 👇
